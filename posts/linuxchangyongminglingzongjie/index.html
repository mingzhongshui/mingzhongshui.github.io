Linux常用命令总结

发表于： 2016-11-10 分类于： 后端
字数： 481 阅读：≈ 1分钟 浏览：

Linux常用命令总结

用户操作

  1. 添加用户组 groupadd

  2. 删除用户组 groupdel

  3. 添加用户 useradd 设置选项 用户名

  4. 指定用户缺省所属组名 useradd -g

  5. 指定用户所属多个组 useradd -G gropu1,group2 user

  6. 指定描述信息 useradd -c

  7. 指定用户失效时间 useradd -e

  8. 修改用户信息 usermod -g group1 user

  9. 删除用户 userdel -r user

文件操作

  1. 创建文件 touch FileName

  2. 删除文件 rm -f

  3. 修改文件权限 chmod xxx FileName

  4. 压缩文件 tar cvf FileName.tar DirName

  5. 解压文件 tar xvf FileName.tar

  6. 复制文件 cp file1 file2

  7. 复制文件连带属性 cp -p file1 file2

  8. 复制链接文件 cp -d file1 file2

  9. 移动文件 mv -r file1 file2

  10. 查看文件 cat

  11. 分页查看 more

  12. 查看文件类型 ls -l

  13. 查找文件 locate

目录操作

  1. 切换目录 cd

  2. 宿主目录 cd ~

  3. 上次目录 cd -

  4. 上级目录 ..

  5. 创建目录 mkdir

  6. 删除目录 rm -f

  7. 递归建立目录 mkdir -p dir1/dir2/dir3

  8. 复制目录 cp -r /etc/test/

  9. 复制目录下所有文件 cp -r /etc/./test/

  10. 移动目录 mv olddir newdir

数据库操作

系统操作

  1. 软连接 ln -s

  2. 硬链接 ln

  3. 查看网络状态 netstat

  4. 查看tcp端口 netstat -t

  5. 查看udp端口 netstat -u

  6. 监听端口 netstat -l

  7. 以IP和端口号显示 netstat -n

  8. 设置系统语言 LANG= 默认英语

其它操作

  1. 帮助命令 man

  2. 给用户发信息 write

  3. 查看当前登陆用户列表 who

  4. 查看IP地址 ping

  5. ping几次 ping -c num IP

  6. 关机 shutdown -h now

  7. 补全命令 tab

  8. 清屏 clear or ctrl+l

  9. 清除一行 clear -u

  10. 命令历史 history

  11. 执行历时第n个命令 !n

  12. 添加别名 alias copy=cp copy复制文件

  13. 删除别名 unalias copy

  14. 查看系统时间 date

  15. 输出重定向 > or >>

  16. 错误输出重定向 2> or 2>>

  17. 管道 | 命令1|命令2|命令3
