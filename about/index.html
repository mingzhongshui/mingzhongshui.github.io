关于我

个人

93 年，北方人，出生在炎热的夏季，双子座 boy。
机械制造专业全日制大专毕业，后转行到互联网，至今已在 IT 行业工作 8 年（目前待业中，寻找合适机会）。

技术栈

  • 主攻语言：PHP
  • 其他语言：Python、Java、JavaScript（早年写过）
  • 前端能力：HTML + CSS 非主要方向，能满足基本协作需求
  • 常用工具/框架：Laravel/Lumen、Yii、CI、PhalApi、Scrapy-Redis、Flask、RabbitMQ、Redis、MySQL、MongoDB、Docker、Git

技术标签：PHP Python Java Redis RabbitMQ Scrapy MongoDB Lumen

工作经历

深圳某上市地产公司（2021.9 – 2024.6）

负责 100+ 租户的 SaaS 系统维护，参与系统微服务拆分、Java 模块重构、技术文档编写。
技术栈：Yii2 + RabbitMQ + Redis + Java

物业管理系统

  • 维护多租户物业系统，累计修复客户反馈 bug 百余个，降低月均 bug 数 30% 以上；
  • 主导多租户开户项目迁移，重新设计开户流程，效率提升 90%，开户时长缩短至分钟级；
  • 负责隐私数据改造项目，完成代码及历史数据改造，确保符合 GDPR 规范，并通过安全审计；
  • 研究多种语言加解密算法，实现帆软报表与 Java 代码间数据加密传输，保障数据安全，支持报表准确生成；
  • 参与物业报修模块 Java 重构，完成基础数据改造及部门对接，保证数据按时交付，支持新业务上线。

深圳某跨境电商公司（2019.12 – 2021.9）

负责公司内部 ERP 系统开发，搭建 CI/CD 环境，自学 Python 研究反爬虫策略，完成亚马逊爬虫项目。
技术栈：PHP + ThinkPHP + Python + Redis

亚马逊平台爬虫系统
纯 Python 爬虫，负责反爬机制研究、架构设计及技术落地。主要包括前台爬虫（商品、评论）和后台爬虫（报表自动下载）。

  • 使用 Scrapy-Redis 作为爬虫框架，Flask 提供接口，Celery 处理并发请求；
  • 破解亚马逊反爬策略（动态代理、请求特征混淆），数据采集成功率 > 98%；
  • 基于 Selenium 实现后台报表自动下载，人力成本降低 70%；
  • Supervisor 集群管理 + MongoDB 分片存储，数据入库效率提升 50%。

公司内部 OA 系统
从单一 ERP 系统拆分为多个子系统（ERP、ECP、FAS、WMS 等），参与拆分方案设计，负责 ECP 系统的迁移工作。

  • 优化大批量数据导出，支持异步导出 百万级 数据；
  • 爬虫业务对接，基于 Lumen 6.0 开发分布式任务系统，Redis 队列 + 多进程消费提速 300%；
  • 对接亚马逊 SP-API，实现订单、广告数据实时分析。

深圳某电商公司（2019.4 – 2019.12）

自研社交 APP，独立完成订单、商品、库存、抢购模块；后期负责交易类 APP 所有后端接口。
技术栈：PHP + Laravel/Lumen + Redis

某交易类 APP

  • 参与需求讨论与架构选型，基于 Lumen 6.0 完成接口编码及表结构设计；
  • 交易异步处理：将交易操作写入 Redis 消息队列，支持高并发，避免数据库宕机；
  • 缓存核心交易商品数据，提升接口响应速度。

某社交类 APP

  • 基于 Lumen 5.8 完成商城模块接口开发，设计相关表结构；
  • 使用 Redis 键空间通知处理过期订单，编写订单结算脚本，利用定时任务实现结算功能；
  • 开发限时抢购模块，设计缓存更新方案，缓存抢购商品数据。

深圳某通信网络公司（2016.4 – 2019.4）

深圳软件协会成员企业。主要负责 OA 办公、自研商城系统的开发，担任小组长。
技术栈：PHP + CI + Redis + MySQL

自研商城平台

  • 独立完成数据库设计，使用 Xmind 绘制前后台架构、模块分工导图；负责购物车、订单、支付模块开发；
  • 基于 RBAC 模型设计后台权限模块，提升不同角色的页面访问控制能力；
  • 后期优化业务流程，优化慢 SQL，图片接入七牛云，热数据存入 Redis。

报餐点餐系统

  • 确定项目架构及数据库设计，监控开发进度，Review 同事代码；
  • 消费模块采用 Workerman 消息推送框架，解决数据实时刷新和异步传输问题；
  • 后期带领团队完成系统升级，增加单台消费机独立消费功能，获客户好评。
